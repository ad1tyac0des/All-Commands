Git Commands

- Cheatsheet

Managing files locally on git

git init

Initializes a new Git repository in the current directory.

git add filename

To add new file in repository. You can use "." for selecting all files

git status

Displays the state of the working directory and the staging area, showing changes that have been staged, changes that have not been staged, and untracked files.

git log

It shows the commited file

git branch

To check the number of branch

git branch branchName

To add a new branch

git switch branchName

To merge one branch with another

Note - For this we have to switch on main(default branch) then only we have to merge.

git branch -d branchName

To delete the branch

git stash

This command will execute when we write code but we dont waana to add nor delete and move to another branch (it store the code in memory).

git stash apply

This command will execute when we want to apply that stored data.

git stash clear

To add new branch

git reset --hard HEAD~1

To move on previous saved point

rm -rf .git

To restart git
